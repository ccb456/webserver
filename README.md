# WebServer 压力测试报告

## 🖥️ 测试环境

| 项目 | 参数 |
|------|------|
| 操作系统 | **Windows 11 x64 (主机端)** |
| 测试工具 | **ApacheBench (ab 2.3)** |
| 客户端平台 | Windows PowerShell |
| 服务器平台 | Ubuntu 22.04 虚拟机 (VMware) |
| 服务器 IP | 192.168.111.100 |
| 端口 | 8080 |
| CPU | 4 核 |
| 内存 | 4GB |
| 线程池配置 | min = 8，max = 16 |
| Keep-Alive | 开启（`-k`） |

---

## ⚙️ 测试命令模板

```bash
.\ab -n <请求总数> -c <并发数> -k http://192.168.111.100:8080/

```

## 📈 测试结果记录
测试 1
```bash
.\ab -n 1000 -c 50 -k http://192.168.111.100:8080/
```

测试 2

测试 3

测试4

## 📊 测试结果汇总

| 编号 | 请求总数 (-n) | 并发数 (-c) | 失败请求数 | 平均响应时间 (ms) | RPS (请求数/秒) | 备注 |
|------|----------------|--------------|--------------|--------------------|------------------|------|
| T1 | 1000 | 50 | 0 | 5.88 | 8501.02 | ✅ 稳定通过 |
| T2 | 10000 | 200 | 0 | 20.84 | 9598.52 | ✅ 稳定通过，响应时间波动小（标准差 8ms），长尾请求控制佳 |
| T3 | 20000 | 500 | 0 | 78.49 | 6370.02 | ✅ 稳定通过，高并发下响应略有延长，存在少量长尾请求（最长 1305ms） |
| T4 | 50000 | 1000 | 44247 | —— | —— | ❌ 测试失败，仅完成 5753 次请求，剩余请求因远程主机强迫关闭连接失败 |
